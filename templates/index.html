Welcome to Mafia schedule database

Here is a list of API endpoints:

  • Get all schedules.

    Returns the list of all schedule names in JSON form.

    Example: {{ base_url }}schedule

  • Get schedule by ID.

    Returns schedule by ID in JSON form. If schedule is not found HTTP Error 404 is returned.

    Example: {{ base_url }}schedule/vawaca2021_seats

  • Find schedules by parameters: players, tables, distance.

    Returns the list of schedule names that match a query (players, tables, distance).

    Example 1: {{ base_url }}find?players=36

    Example 2: {{ base_url }}find?players=36&tables=3&distance=10
